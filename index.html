Proyectos

Cambio climatico

Estamos trabajando enconjuntos de datos históricos y actuales para desarrollar modelos predictivos que proyectaran los futuros impactos del cambio climático en la región del Pacífico.
Empleando herramientas como Pandas, NumPy y scikit-learn, limpiamos y analizamos datos de temperatura, precipitaciones y emisiones de CO2. Creamos visualizaciones con Matplotlib y Seaborn para comunicar nuestros hallazgos de manera clara y efectiva, destacando las áreas más vulnerables y proponiendo estrategias de mitigación.

Violencia rural

Utilizando análisis de datos para identificar patrones y factores contribuyentes. Trabajamos con datos de incidentes de violencia, características socioeconómicas y geográficas de las comunidades afectadas.

Utilicé Pandas para limpiar y organizar los datos, y NumPy para realizar análisis estadísticos. Desarrollamos modelos predictivos con scikit-learn para identificar las áreas con mayor riesgo de violencia y propusimos intervenciones basadas en nuestros hallazgos.
